# Informe final — Programación 1

El informe completo en PDF se entrega por correo. Este resumen documenta el alcance del proyecto, las decisiones técnicas, el uso de PEP 8 y PEP 257, la modularización, el entorno virtual, las dependencias, el Zen de Python, la defensa del desarrollo, la investigación sobre visión por computadora, el uso reflexivo de IA y las limitaciones metodológicas.

El programa no realiza diagnóstico médico ni predice lesiones. Detecta puntos corporales y estima métricas cinemáticas 2D para apoyar una observación exploratoria del salto.
